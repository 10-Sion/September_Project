지시 요구사항

	메인 - 로그인 => 서브페이지 이동		로그인 시 id, pw 찾기 필요 X
	서브1 - 마이 페이지	=> 자기소개 입력 칸

	서브2 - 강의실 페이지 (전체 강의 목록)
		강의실 페이지 => 세부 강의 페이지 홈: 서브3

	서브3 - 세부 강의 =>    강의 정보	-> 수업 계획서
			      수업 활동	-> 강의 리스트 -> 강의 창
			      커뮤니티	-> 공지사항, 과제 제출, Q&A // 게시판 형식


요구사항 구조 따른 수정안

	메인 페이지 	헤더: *강의, *공지사항 네비게이션 추가
			메인: *강의 목록 => 슬라이더 or 스와이퍼의 사용
			       *공지 사항 => 모든 교수가 업로드한 공지 목록 표시 (*테이블 추가)

	서브 페이지	본인 정보 페이지 => 기존 정보 표시 유지, *자기소개 입력 칸 추가

			강의 리스트 페이지 => 기존 형태 유지 (강의 리스트 내 클릭 - 강의 세부 정보 페이지)

				강의 세부 정보 페이지 =>	좌측 네비게이션 메뉴 변경 (강의 정보, *수업 활동, *커뮤니티)
					페이지 구성 =>	메인 / 공지사항 / 강의 리스트

					수업 계획서 =>	수업 계획, *평가 계획
					*강의 창	   =>	해당 강의에 대한 영상을 *iframe으로 볼 수 있도록
					*과제 제출  =>	*파일 첨부 가능한 과제 제출 게시판
					*공지 사항  =>	글의 작성(교수) 및 조회(학생)
					*Q&A	   =>	공지사항 페이지 동일 / 테이블 차이 두도록
